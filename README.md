## Community-calculator Client
+ [Live](https://community-calculator.now.sh/)  
+ [API Repo](https://github.com/DavidHaugen/community-calculator-api)  

### Summary
Community Calculator is a simple calculator app with live updates using socket.io. Any calculations made on the site will be displayed live to all users currently accessing the calculator. The result log is limited to the ten most recent calculations.  

### Tech Stack
+ Built with React and Socket.io
+ Deployed with ZEIT Now
